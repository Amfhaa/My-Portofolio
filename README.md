## 🎨 Architectural & Design Philosophy

Unlike conventional, hyper-polished minimalist webs, this portfolio intentionally embraces the **Neo-Brutalism** movement. Key design choices include:
*   **High-Contrast Geometric Shapes:** Sharp borders (`box-shadow` without blurs) and stark outlines.
*   **Asymmetric Layouts:** Breaking traditional grid rules to create visual tension and engagement.
*   **Unapologetic Typography:** Heavy weights and structured hierarchies designed for maximum readability.
*   **Raw Color Palette:** Vibrant, saturated accents juxtaposed against structural dark lines.

## 🛠️ Built From Scratch (Vanilla Tech Stack)

To maintain absolute control over performance, layout manipulation, and layout rendering, this site is built entirely without heavy bloated frameworks:

*   **HTML5** – Strict semantic markup ensuring optimal SEO and accessibility structure.
*   **CSS3 (Custom Variables & Grid/Flexbox):** Architected using native CSS grid patterns, hard drop-shadows (`solid box-shadow`), and robust media queries for flawless responsiveness.
*   **JavaScript (ES6+)** – Lightweight vanilla scripting driving dynamic micro-interactions, dark/light theme switching (if applicable), and layout shifts.
*   **Vercel Production Edge:** Hosted via Vercel for instant caching, global CDN routing, and optimized loading speeds.

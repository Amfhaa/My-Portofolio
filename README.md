## 🇺🇸 English Version

### ✨ Features
* **⚡ Bilingual Toggle (ID/EN):** Fully supports native translation switching between Indonesian and English for global reach.
* **🎵 Integrated Media Control:** Features an interactive background music controller (Play/Pause) built into the layout.
* **🎨 Neo-Brutalism Aesthetic:** High-contrast color palette, asymmetrical elements, raw typography, and hard drop-shadows.
* **📱 Fully Responsive:** Fluidly adapts to everything from tiny mobile viewports to large desktop monitors.

### 🛠️ Built With (Vanilla Stack)
To maintain absolute layout precision and high-speed loading times, this project avoids heavy framework overhead:
* **HTML5** – Structured semantic markup.
* **CSS3** – Custom variables, hard solid `box-shadow` rules, Flexbox, and Grid layouts.
* **Vanilla JavaScript (ES6)** – Handles the language switching mechanism, audio playback logic, and dynamic UI states.
* **Vercel** – Global production hosting and edge caching.

### 🚀 Local Setup
1. Clone the repository:
   ```bash
git clone [https://github.com/amfhaa/amfhaa.github.io.git](https://github.com/amfhaa/amfhaa.github.io.git)

```
2. Open `index.html` directly in your browser or run it via a local environment like VS Code's **Live Server**.

## 🇮🇩 Versi Bahasa Indonesia

### ✨ Fitur Utama
* **⚡ Pengubah Bahasa (ID/EN):** Mendukung fitur *bilingual* penuh yang memungkinkan pengunjung beralih antara Bahasa Indonesia dan Bahasa Inggris secara instan.
* **🎵 Kontrol Musik Latar:** Dilengkapi dengan tombol interaktif (Play/Pause) untuk mengontrol pemutaran musik chiptune di latar belakang.
* **🎨 Estetika Neo-Brutalism:** Karakter visual yang kuat memanfaatkan garis batas tegas, bayangan solid (`box-shadow` tanpa blur), dan warna-warna saturasi tinggi.
* **📱 Desain Responsif:** Layout fleksibel yang nyaman diakses dari perangkat hp pintar hingga layar komputer.

### 🛠️ Teknologi yang Digunakan
Portofolio ini dibangun murni menggunakan teknologi *front-end* dasar (*vanilla code*) untuk menjamin performa pemuatan yang super cepat tanpa *overhead* dari framework besar:
* **HTML5** – Penulisan konten semantik dan terstruktur.
* **CSS3** – Menggunakan CSS Variables, layouting Flexbox/Grid, serta manipulasi *hard shadow*.
* **Vanilla JavaScript (ES6)** – Mengatur logika perpindahan bahasa teks secara dinamis, kontrol audio musik, dan interaksi komponen.
* **Vercel** – Infrastruktur *hosting* produksi massal yang andal.

### 🚀 Menjalankan Secara Lokal
